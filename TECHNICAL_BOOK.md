# 物联网智能生成与编译平台 (IoT Smart Generation Platform)
## 项目技术书与架构说明文档 (Technical Specification & Architecture Book)

本技术书基于用户最新需求分析进行设计。平台定位为 **一站式物联网智能系统生成与在线编译器平台**，旨在通过大语言模型（LLM）双脑架构，将普通开发者的自然语言创意一键转化为具有工业级规范的 Arduino 固件代码、接线逻辑 CAD 拓扑与部署文档。

---

## 一、 系统愿景与核心定位

传统物联网开发存在“温控、硬件、通讯、屏幕”四大领域碎片化严重、代码驱动与物理引脚接线易冲突等痛点。
本平台定位于 **“设计即代码 (Design as Code)”**：
1. **创意输入与提炼**：通过低门槛日常语言，捕捉用户感知（传感器）、交互（小屏）、反馈（继电器蜂鸣器）、上报（MQTT/HTTP）意图。
2. **硬件总线引脚自动标定 (Pin Auto-Allocation)**：基于目标板芯片平台（ESP32 或 STM32）的总线限制，自动完成无冲突的引脚物理映射。
3. **固件级 C++ 源码包生成**：直接生成无空函数、附带详尽注释、可立即编译的 `main.cpp` 与 `pins.h` 工程。
4. **CAD 交互接线图渲染**：动态渲染矢量接线拓扑与色彩区隔线束，支持一键导出 SVG。

---

## 二、 平台技术架构设计 (System Architecture)

系统采用全栈式前后端分离及多大模型微积分驱动设计：

### 1. 前端架构 (Client-Side Platform)
- **核心框架**：React 18 + Vite (极致敏捷，秒级加载反馈)
- **视觉风格**：基于 **Swiss / Modern Minimalist** 白底极简设计风格，大 negative space，高反差 Inter 字体配 JetBrains Mono。
- **动效库**：Framer Motion (用于步骤跳转、模型加载活泼微动效)
- **图标包**：`lucide-react` 自主矢量图标库
- **CAD 渲染引擎**：SVG 扁平电路拓扑动态生成，保障高缩放比清晰度

### 2. 后端服务端架构 (Backend API Middleware)
- **开发语言**：Node.js + TypeScript
- **框架支持**：Express.js 路由层 + Vite 开发服务器路由中间件
- **编译打包**：采用 `esbuild` 快速生成自包含单文件 `dist/server.cjs`，脱离严格的 Node 路径运行时开销。

### 3. 多模型协作编译引擎 (Dual-Model Compiler Engine) [新增]
支持用户按需进行编译模型切换：
- **Gemini 3.5 Flash**：多模态支持优秀，代码及提示词提炼响应极速。
- **DeepSeek V3 / Coder**：聚焦深度代码推理，可精细编制引脚兼容逻辑，无残留伪代码，输出结构高度对齐。

### 4. 数据持久化 (Storage Layer)
- 搭载 **Firebase Firestore** 分布式非关系型数据库。
- 项目加载、历史查阅和个人生成方案进行云端密道同步存贮，离线时降级为本地 `localStorage`，保障无中断开发。

## 三、 物联网智能生成完整工作流 (Platform Workflows)

```
[步骤 1: 需求输入] --> [步骤 2: 方案提炼 (优化配置)] --> [步骤 3: 锁脚与代码生成] --> [步骤 4: 配线 CAD 与打包]
```

1. **步骤 1：主芯片 & 引擎选择**
   - 开发者选定 ESP32 / STM32 系列芯片。
   - 一键自选 **Gemini 3.5** 或 **DeepSeek V3** 生成编译引擎。
2. **步骤 2：优化提示词提炼（优化阶段）**
   - 原始需求被转译为包含主控、感知、交互、报警、供电在内的 JSON 元数据。
   - 产出中英文系统特征方案，允许开发者在线微调。
3. **步骤 3：固件及线缆生成（生成阶段）**
   - 结合硬件配置表，调用 AI Compiler 自动匹配相应片上资源（如 I2C 硬件组、GPIO 电平）。
   - 生成符合系统规范的代码。
4. **步骤 4：CAD 二维可视呈现与 ZIP 下载**
   - 导出 `wiring_diagram.svg` 硬件连线图。
   - 一键将 `main.cpp`, `pins.h`, `README.md` 通过 `jszip` 打包为二进制 ZIP 格式包。
